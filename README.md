# 健康提醒助手

一个高端精简的Windows桌面小软件，用于久坐提醒、喝水提醒和护眼提醒。

## 功能特点

- 🧘 **久坐提醒** - 自定义时间间隔，提醒起身活动
- 💧 **喝水提醒** - 定时提醒保持水分摄入
- 👀 **护眼提醒** - 遵循20-20-20法则，保护眼睛
- 📊 **统计面板** - 记录每日休息次数、喝水次数、工作时长
- 🔔 **提示音** - 可开关的提示音效
- 🚀 **开机自启** - 支持开机自动运行
- 📌 **托盘隐藏** - 关闭窗口最小化到托盘，后台继续运行
- ⚡ **低资源占用** - 基于Tauri，内存占用极低

## 技术栈

- **Tauri 2.0** - Rust后端，资源占用极低
- **Vite** - 快速的前端构建工具
- **原生HTML/CSS/JS** - 无框架依赖，极致轻量

## 开发环境要求

- Node.js 18+
- Rust 1.70+
- Windows 10/11

## 安装依赖

```bash
npm install
```

## 开发运行

```bash
npm run tauri dev
```

## 构建发布

```bash
npm run tauri build
```

构建完成后，可执行文件位于 `src-tauri/target/release/` 目录。

## 使用说明

1. 启动后显示主界面，可以看到倒计时和各项设置
2. 点击各提醒项的开关可以启用/禁用对应提醒
3. 修改时间间隔后自动保存
4. 点击窗口关闭按钮会最小化到系统托盘
5. 双击托盘图标可以重新显示窗口
6. 右键托盘图标可以选择"显示主窗口"或"退出"
